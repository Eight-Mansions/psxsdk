This contains a beta tool from SCEE for manipulating and
texturing RSD objects: 

	FILE		VERSION
	---------------------------
	rsdtool.exe 	Beta 0.8

==================
INSTRUCTIONS:
Place the file in your \ps\psxgraph\bin
folder, such as in the following command:

	move rsdtool.exe c:\ps\psxgraph\bin

assuming that the "c:\ps" directory is where you've installed your
development system.  This is a very early beta.
